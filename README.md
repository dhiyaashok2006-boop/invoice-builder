# Invoice Builder — React P2

A responsive invoice generation app built with React JS, TailwindCSS, and React Hooks.

## Features

- Enter client & sender details (name, email, address)
- Add/edit/delete multiple line items (description, quantity, unit rate)
- Auto-calculates subtotal, tax, and grand total in real-time
- Editable invoice number, date, due date, and tax rate
- Multi-currency support (INR, USD, EUR, GBP)
- Export invoice as a downloadable PDF (jsPDF + html2canvas)
- Print-friendly layout
- Fully responsive (mobile + desktop)

## Tech Stack

- React JS (v18)
- TailwindCSS
- React Hooks (`useState`, `useCallback`, `useRef`)
- jsPDF + html2canvas (PDF export)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/invoice-builder.git
cd invoice-builder
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

This creates an optimized build in the `build/` folder.

## Deployment (Netlify)

### Option 1 — Netlify Drop (Easiest)

1. Run `npm run build`
2. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
3. Drag and drop the `build/` folder
4. Your app is live instantly!

### Option 2 — Netlify CLI

```bash
npm install -g netlify-cli
npm run build
netlify deploy --prod --dir=build
```

### Option 3 — Netlify via GitHub

1. Push your code to GitHub
2. Go to [https://app.netlify.com](https://app.netlify.com)
3. Click "New site from Git"
4. Select your repository
5. Build command: `npm run build`
6. Publish directory: `build`
7. Click "Deploy site"

## How It Works

- All invoice data is managed in React local state (`useState`) in `App.jsx`
- `InvoiceForm` handles all user input and passes changes via callback props
- `InvoicePreview` renders the printable layout and triggers PDF export
- PDF is generated by capturing the invoice DOM node with `html2canvas`, then embedding it into a jsPDF document
